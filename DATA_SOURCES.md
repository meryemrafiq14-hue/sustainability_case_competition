# Data Sources

This project uses public or anonymized publication metadata from university sources.

## Fields used
- Author name
- Department
- Keywords
- Abstract
- SDG labels (top 1, top 2, top 3)
- Publication year

## Derived fields
- Primary SDG (most frequent)
- Research method (keyword inference)
- Career stage (years since first publication)
- Top keywords (frequency ranking)

## Demo transparency
Some user search inputs are varied to simulate different scenarios.  
All scores and explanations are algorithmically generated.
