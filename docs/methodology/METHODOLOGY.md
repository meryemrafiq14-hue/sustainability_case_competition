# Methodology Summary

## Pipeline
Publication CSV → researcher profiles → CCS demo data → Power BI

## Scoring
- Topic match (45%): hierarchical SDG overlap logic
- Method match (40%): complementary methods score higher
- Career fit (15%): mentorship/peer balance
- CCS = weighted sum

## Limitations
- Demo data includes simulated user search inputs
- Scores are directional, not predictive
