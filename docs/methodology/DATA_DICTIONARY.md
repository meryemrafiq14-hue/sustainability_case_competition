# Data Dictionary (CCS_Demo_Data.csv)

| Column | Meaning |
|---|---|
| UserName | Researcher initiating the search |
| UserDept | Department of searching researcher |
| UserSDG | Primary SDG of searching researcher |
| UserMethod | Inferred research method |
| UserStage | Career stage |
| MatchName | Suggested collaborator |
| MatchDept | Department of match |
| MatchSDG | Match's primary SDG |
| MatchMethod | Match's method |
| MatchStage | Match's career stage |
| Topic_Match | SDG alignment score |
| Method_Match | Method complementarity score |
| Career_Fit | Career stage alignment score |
| CCS_Total | Weighted total score |
| Explanation | Generated summary of why the match fits |
